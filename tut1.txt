# Tutorial1

Q1: How can you list all files in a directory, with the most recent one displayed last?
Ans: ls -tr

Q2: How can you display the first 25 lines of a file? How can you display just lines 16 through 25?
Ans: head -25 <file name>
     head -25 <file name> | tail -10
Q3: How can you search for all commands relating to python?
Ans: man python
